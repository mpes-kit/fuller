fuller.utils.trim_2d_edge(arr, edges, axes=0, 1)¶

Trim 2D edges in the first two dimensions of an nD array.

Parameters
arrnumpy array

Array to trim .

edgesnumeric/list/tuple/numpy array

The amount of edges to trim. If a single value is assigned, the two ends of the axes are trimmed equally. If a list of four different values is assigned, they are applied to the two axes in the order (start_1, end_1, start_2, end_2).

axeslist/tuple

Specified axes/dimensions to trim.

Return
trimmednumpy array

Axis-trimmed array.
